# WEBSITE KEPEGAWAIAN

Repositori ini merupakan pusat pengembangan sistem informasi manajemen kepegawaian berbasis web yang disusun oleh Kelompok 6 sebagai proyek tugas besar dalam mata kuliah Desain dan Pemrograman Web. Melalui repositori ini, kami mendokumentasikan setiap tahapan pengembangan, mulai dari perancangan basis data hingga implementasi antarmuka pengguna, guna menjamin transparansi dan kualitas kode yang sesuai dengan standar industri."
